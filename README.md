# Gantt

A lightweight, URL-first Gantt chart tool. Build a timeline in seconds and share it instantly via link — no login, no setup, no friction.

Perfect for sharing project timelines with clients or teammates.

## How it works

Everything lives in the URL. Create your chart, copy the link, and share it. That's it.
